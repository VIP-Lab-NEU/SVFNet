# SVFNet: Multi-modal Scene-level Classification

SVFNet is a multi-modal neural network designed for scene-level classification, leveraging both RGB and Sonar dual-stream inputs. It features a MAXVIT-based dual backbone, Global Query Space Attention (GQSA), and supports advanced visualization techniques like Grad-CAM and t-SNE.

## 📋 Features
- **Dual-Stream Architecture**: Processes RGB and Sonar images simultaneously.
- **Global Query Space Attention (GQSA)**: Enhances feature representation across modalities.
- **Comprehensive Evaluation Tools**: Built-in support for PR curves, Confusion Matrices, t-SNE clustering, and Grad-CAM heatmaps.
